# 1. Introduction

The task is to implement a classic web-based online chat application with support for:
- user registration and authentication
- public and private chat rooms
- one-to-one personal messaging
- contacts/friends
- file and image sharing
- basic moderation and administration features
- persistent message history
The application should represent a typical “classic web chat” experience, with
straightforward navigation, room lists, contact lists, message history, notifications, and
online presence indicators.
The system is intended for moderate scale and should support up to 300 simultaneously
connected users.
